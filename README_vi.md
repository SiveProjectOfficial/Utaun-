# 【Utaun là gì?】
Phần mềm Windows tạo voice bank CV cho UTAU bằng tổng hợp cộng tính. Tự động tạo tệp ZIP trong thư mục Documents.

### 【Tính năng】
* Tự động tạo ZIP trong Documents.
* Tạo nguyên âm (a, i, u, e, o, n) bằng tổng hợp cộng tính.
* Kết hợp với phụ âm bên ngoài (hàng Ka đến Pa).
* Hỗ trợ alias Hiragana và Romaji.
* Tự động tạo `oto.ini`, `character.txt` và `readme.txt`.

### 【Điều khoản】
Nghiêm cấm sửa đổi, dùng cho mục đích thương mại hoặc phân phối lại phần mềm. Bạn có quyền tự quyết định quy định cho voice bank của mình.
