# 【什么是 Utaun？】
这是一款仅通过完全无生物的加算合成技术来生成 UTAU 专用 CV 音源的 Windows 软件。它会在您的“文档”文件夹中自动生成音源 ZIP 文件。

### 【特点】
* 运行后在“文档”中自动生成音源 ZIP。
* 通过完全无生物的加算合成生成元音（a、i、u、e、o、n）。
* 通过与外部子音行（Ka 行至 Pa 行）组合生成 CV 音源。
* 内部生成 m / n / y / w 行。
* 支持平假名别名 + 英语罗马字别名（发音均为日语）。
* 自动生成 `oto.ini` / `character.txt` / `readme.txt`。
* 可将 PNG 图片转换为 JPG。
* 为防止乱码，`readme.txt` 和 `character.txt` 以 Shift_JIS (ANSI) 编码输出。
* 可修改基频（音高）。
* 可自由设置音源名称（角色名）。

### 【适用场景】
* 想要用无生物的声音制作 UTAU 音源时。
* 为不擅长录制自己声音的人制作音源。
* 加算合成语音的素材研究。
* 自制音源的原型设计。
* 解压 ZIP 后直接作为“人力 Vocaloid”素材使用。
* 作为音 MAD、YTPMV 等视频制作的素材。

### 【Utaun 使用方法】
1. 下载 `Utaun.ver-1.0.4.zip`。（请从 **Releases** 页面下载。）
2. 解压 `Utaun.ver-1.0.4.zip`。
3. 将 `Utaun.exe` 放置在任意文件夹（推荐放置在桌面）。
4. 启动 `Utaun.exe` 并按照屏幕指示操作。
5. 当显示“（音源名）.zip 已在文档中生成！”时，UTAU 音源即制作完成。

### 【生成的 UTAU 音源使用方法】
* **UTAU**：解压生成的 ZIP，即可获得可直接导入 UTAU 的音源文件夹。
* **OpenUTAU**：可以直接以 ZIP 格式导入（无需解压）。
* *关于各软件的具体使用方法，请参考各自的帮助文档、手册或官方资料。*

### 【运行环境】
* Windows 10–11 (64bit)

### 【尚不支持】
* 浊音元音（あ゙、い゙、ゔ、え゙、お゙）
* 气声、呼吸声等表情差分。
* 拗音（kya、kyu、kyo / sha、shu、sho 等）。
* 连续音 (VCV)。
* CVVC。
* 其他特殊发音。

### 【利用条款】
Utaun 软件本体 (`Utaun.exe`) 禁止以下行为：
* 加工 / 编辑 / 改造
* 商用利用
* 再发布
* 反编译（拆解）
