# 【Apa itu Utaun?】
Perangkat lunak Windows untuk menghasilkan voice bank CV UTAU melalui sintesis aditif. Membuat file ZIP secara otomatis di folder Documents.

### 【Fitur】
* Pembuatan ZIP otomatis di folder Documents.
* Menghasilkan vokal (a, i, u, e, o, n) dengan sintesis aditif.
* Bisa digabung dengan konsonan eksternal (baris Ka hingga Pa).
* Mendukung alias Hiragana dan Romaji.
* Otomatis membuat `oto.ini`, `character.txt`, and `readme.txt`.

### 【Ketentuan】
Dilarang memodifikasi, menggunakan secara komersial, atau mendistribusikan ulang perangkat lunak. Voice bank yang dihasilkan adalah milik Anda.
