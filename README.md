【Utaunとは？】  
完全無生物の加算合成だけで UTAU 用の CV 音源を生成する Windows ソフトです。  
ドキュメントに音源 ZIP を自動生成します。  

【特徴】   
 ・実行するとデスクトップに音源 ZIP を自動生成  
 ・完全無生物の加算合成による母音生成（あ・い・う・え・お・ん）  
 ・外部子音行（か行〜ぱ行）との組み合わせで CV 音源を生成  
 ・m / n / y / w 行の内部生成  
 ・ひらがなエイリアス + 英語ローマ字エイリアスに対応（発音はすべて日本語）  
 ・oto.ini / character.txt / readme.txt を自動生成します  
 ・PNG 画像を JPG へ変換できます  
 ・readme.txt と character.txt は文字化け防止のため Shift_JIS（ANSI）で出力しています  
 ・基本周波数（声の高さ）を変更できます  
 ・音源名（キャラクター名）を自由に設定できます  

【想定用途】  
 ・無生物の声で UTAU 音源を作りたい時  
 ・地声で録音するのが苦手な人向けの音源制作  
 ・加算合成音声の素材研究  
 ・自作音源のプロトタイピング  
 ・ZIP を解凍して、そのまま人力ボカロ素材として使う用途  
 ・音MAD・YTPMV などの素材としての用途  

【Utaunの使用方法】  
 ① Utaun.ver-1.0.0.zip をダウンロードします。（Releasesからダウンロードして下さい。）  
 ② Utaun.ver-1.0.0.zip を展開（解凍）します。  
 ③ Utaun.ver-1.0.0.exe を任意のフォルダ（デスクトップ推奨）に置きます。  
 ④ Utaun.ver-1.0.0.exe を起動し、画面の指示に従います。  
 ⑤ 「（音源名）.zip がデスクトップに出来上がったよ！」と表示されたら UTAU 音源の完成です。  

【生成したUTAU音源の使用方法】  
 ・UTAU  
   生成された ZIP を解凍すると、そのまま UTAU に読み込める音源フォルダが展開されます。  

 ・OpenUTAU  
   ZIP のまま読み込めます（解凍不要）。  

 ・UTAlet  
   ZIP のまま読み込めます（解凍不要）。  

 ※ 各ソフトでの具体的な利用方法は、それぞれのヘルプ・マニュアル・公式ドキュメントをご参照ください。

What is Utaun?
Utaun is a Windows application that generates CV voicebanks for UTAU using purely inorganic additive synthesis.
It runs as a standalone executable and automatically creates a ZIP voicebank file in your Documents folder.

Features
Automatically generates a voicebank ZIP on your desktop when executed

Produces vowel sounds (a / i / u / e / o / n) using fully inorganic additive synthesis

Combines synthesized vowels with external consonant sets (ka–pa rows) to create CV voicebanks

Internally generates the m / n / y / w consonant rows

Supports both Hiragana aliases and English romaji aliases (all pronunciations are Japanese)

Automatically generates oto.ini, character.txt, and readme.txt

Can convert PNG images to JPG

Outputs readme.txt and character.txt in Shift_JIS (ANSI) to prevent mojibake

Allows customization of the base pitch (voice height)

Allows free naming of the voicebank (character name)

Intended Use Cases
Creating UTAU voicebanks with inorganic/artificial voices

For users who find it difficult to record their own natural voice

Research on additive-synthesis-based vocal materials

Prototyping your own voicebanks

Using the generated ZIP directly as human‑vocaloid (人力ボカロ) material

Using it as source material for 音MAD / YTPMV production

How to Use Utaun
Download Utaun.ver-1.0.0.zip (from the Releases page).

Extract the ZIP file.

Place Utaun.ver-1.0.0.exe in any folder (Desktop recommended).

Run Utaun.ver-1.0.0.exe and follow the on-screen instructions.

When the message “(VoicebankName).zip has been created on your desktop!” appears, your UTAU voicebank is ready.

How to Use the Generated UTAU Voicebank
UTAU
Extract the generated ZIP file. The extracted folder can be loaded directly into UTAU.

OpenUTAU
You can load the ZIP file directly without extracting it.

UTAlet
You can load the ZIP file directly without extracting it.
