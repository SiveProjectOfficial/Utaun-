# 【Utaun이란?】
완전 무생물의 가산 합성만으로 UTAU용 CV 음원을 생성하는 Windows 소프트웨어입니다. 문서 폴더에 음원 ZIP 파일을 자동 생성합니다.

### 【특징】
* 실행 시 문서 폴더에 음원 ZIP 자동 생성.
* 완전 무생물의 가산 합성에 의한 모음 생성 (아·이·우·에·오·응).
* 외부 자음 행 (카행~파행)과의 조합으로 CV 음원 생성.
* m / n / y / w 행의 내부 생성.
* 히라가나 에일리언스 + 영어 로마자 에일리언스 대응 (발음은 모두 일본어).
* `oto.ini` / `character.txt` / `readme.txt` 자동 생성.
* PNG 이미지를 JPG로 변환 가능.
* `readme.txt`와 `character.txt`는 글자 깨짐 방지를 위해 Shift_JIS(ANSI)로 출력됩니다.
* 기본 주파수(목소리 높이) 변경 가능.
* 음원명(캐릭터 이름) 자유 설정 가능.

### 【상정 용도】
* 무생물의 목소리로 UTAU 음원을 만들고 싶을 때.
* 자신의 목소리로 녹음하는 것이 서툰 분들을 위한 음원 제작.
* 가산 합성 음성 소재 연구.
* 자작 음원의 프로토타이핑.
* ZIP을 압축 해제하여 그대로 인력 보컬로이드 소재로 사용.
* 음MAD·YTPMV 등의 소재로 사용.

### 【Utaun 사용 방법】
1. `Utaun.ver-1.0.4.zip`을 다운로드합니다. (**Releases**에서 다운로드해 주세요.)
2. `Utaun.ver-1.0.4.zip`을 압축 해제합니다.
3. `Utaun.exe`를 임의의 폴더(바탕화면 권장)에 둡니다.
4. `Utaun.exe`를 실행하고 화면의 지시에 따릅니다.
5. "(음원명).zip이 문서 폴더에 생성되었어!"라고 표시되면 UTAU 음원이 완성됩니다.

### 【생성된 UTAU 음원 사용 방법】
* **UTAU**: 생성된 ZIP을 압축 해제하면 UTAU에서 바로 불러올 수 있는 음원 폴더가 나타납니다.
* **OpenUTAU**: ZIP 상태 그대로 불러올 수 있습니다 (압축 해제 불필요).
* *각 소프트웨어에서의 구체적인 사용 방법은 해당 도움말·매뉴얼·공식 문서를 참조해 주세요.*

### 【동작 환경】
* Windows 10~11 (64bit)

### 【미지원】
* 탁음 모음 (아゙·이゙·ゔ·에゙·오゙)
* 속삭임·브레스·숨소리 등의 표정 차분.
* 요음 (캬·큐·쿄 / 샤·슈·쇼 등).
* 연속음 (VCV).
* CVVC.
* 기타 특수 발음.
